4. Работа с программой:

1) Программа защищена от сбоев в большенстве кртических мест и пользователь просто не может "нанести урон" игре, вводя некорректные данные и прочее. Однако есть команды, которые потенциально опасны при неумелом использовании. К ним относятся: команда сброса дампа в файл (если вы введёте имя системного файла, то он будет удалён), запись в память по адресу, копирование блоков памяти.

2) Запуск.
После запуска Евма предложит выбрать версию Героев. После чего нажимаем  Enter.
Далее Евма попытается отыскать процесс, ориентируясь на заголовок окна. Если Герои не запущены или у них изменён заголовок, Евма не найдёт нужный процесс. В любом случае вы можете в любой момент закрыть программу.

3) Команды.
Команды в Евме вводятся одним нажатием определённой клавишы. При запуске выводится список команд. Если вы забыли некоторые, но нажмите "H" для вывода сводки команд.
Рассмотрим параметры каждой из команд:

~> Получить ЕРМ-переменную.
Параметры: *переменная* *кол-во*
(P.S Здесь и далее: параметры следует отделять только ОДНИМ пробелом)
*переменная* - переменная ЕРМ (поддерживаются  глобальные V,W,F (флаги),Z), причём регистр первого символа не важен.
*кол-во* - кол-во считываемых и выводимых на экран перееменных (консоль рассчитана максимально на 1000 строк, при превышении данного кол-ва строки сдвигаются, то есть самые первые будут потеряны). Данный параметр можно опустить, если имеется ввиду вывод только одной переменной.

Пример:
z55
-----------
Выведет значение переменной z55


~> Установить ЕРМ-переменную.
Параметры: *переменная* *новое значение1* *новое значение2* *новое значение 3*...
*новое значение* - корректное для данного типа переменной значение. Если указаны несколько значений, то будут установлены и переменные со индексом на 1,2,3 больше того, что указан в *переменная*.

Пример:
v1 100 200 300
-------------
Установит v1 в 100, v2 в 200, v3 в 300.


~> Заполнить серию переменных указанным значением.
Параметры: *переменная* *кол-во* *чем_заполняем*

Пример:

v1 10000 0
-------------------
Обнулит все v-переменные


~> Выбор героя.
Параметры: *номер_героя*

Пример:

155
-------------------
Выбран Ксерон


~> Чтение данных по адресу.
Параметры: *адрес* *тип* *кол-во*
*адрес* - адрес в десятиричной системе. Если хотите писать в 16-чной, то ставьте перед адресом знак "$"
*тип* - поддерживаются I(4 байта, число), W(2 байта, число), B(1 байт, число), Z(x байт, нуль-конечная строка), C(символ)

Пример:

$887668 i 1
-------------------
Выведет на экран переменную v1


~> Запись данных по адресу.
Параметры: *адрес* *тип* *новое значение1* *новое значение2*...

Пример:

$887668 b -1 -1 -1 -1
-------------------
Установит v1 в -1


~> Получить адрес глобальной ЕРМ-переменной.
Параметры: *переменная*
Пример:

w1
-------------------
Выведет на экран адрес переменной w1 выбранного героя.

!!!!!!!!!!: Герой по умолчанию выбран с номером 0.


~> Установка одной переменной значения другой.
Параметры: *переменная_приёмник* *переменная_источник*

Пример:

v1 w77
-------------------
Выведет на экран переменную v1 получила значения переменной w77 текущего героя.


~> Сбросить дамп переменных в файл.
Параметры1: *имя_файла*
Параметры2: *переменная* *кол-во*

Пример:

z_vars.dmp
z1 10
-------------------
Будет создан файл z_vars.dmp и в него скинуты значения переменных z1-z10 (в нечитаемом виде)


~> Загрузка переменных из дампа.
Параметры1: *имя_файла*
Параметры2: *переменная* *кол-во*

Пример:

z_vars.dmp
z1 10
-------------------
Загрузит из файла z_vars.dmp переменные z1-z10


~> Копирования блока памяти.
Параметры: *адрес_источник* *адрес_получатель* *кол-во байт*

Копирует байты из одной области памяти в другую.

Пример:

$880000 $900000 100
-------------------
Скопирует сто байт из $880000 в $900000


НАСТРОЙКА И РАБОТА С КОНСОЛЬЮ:

1) Правая кнопка по консоли->Свойства
2) Активируем галочку выделение мышью.
3) Жмём ОК. Выбираем "сохранить для всех окон с таким же именем"

Всё, теперь мы можем выделять текст мышью и нажатием ENTER копировать выделенный текст в буфер обмена.

Для вставки текста из буфера обмена правая кнопка мыши на загаловке консоли->Изменить->Вставка.
