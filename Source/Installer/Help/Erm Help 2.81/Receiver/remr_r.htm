Stack Magic Resistance (MR)
 RECEIVER

You may call it ONLY from inside !?MR trigger sections.

!!MR:XXXX; Stack Magic Resistance support

OPTIONS

D?$; Basic (without resistance) damage.
F$; Final (corrected) damage.
M$; Monster type. (See Format C).

Comments:
1. Usually you can use it for getting the number but you can also set the value (make sense only in !?MR0 trigger) to simulate the resistance (actually there is only golems resistance and Elementals sensitivity).
2. This may be got or set. Both make sense. If you change this in !?MR0, your value will be used as a basic spell damage for the creature by resistance calculating routine.
3. If you check it in !?MR0, it will be equal to D$ value. Setting this value in !?MR0 makes no sense for this value will be set by resistance calculating routine. Though it is very useful to be set in !?MR1. Using this way you can ignore a default WoG resistance and set your own value that will be returned as a final damage applied to the creature stack.
N?$; Stack number on the battlefield (0...41).

Comments:
You can only get this value. There is a VERY IMPORTANT thing here! The AI calculates losses without real battle if no human player takes part in the battle. Moreover, the AI calculates possible battles every AI hero step to estimate whether it may attack a human hero or not. This means that this command will work ONLY if there is a real battle (even if it is a quick battle, it IS a real battle). You know that we have flag 1000 in !?BA trigger to determine a real battle. So all that was said about !?BA trigger is identical for !?MR trigger. Anyway, there is a check and you will get a syntax error if you try to use N$ command not in the real battle.


Note:
This command returns invalid numbers.
A workaround script for WoG. version 3.58f:
S?$; Spell number. (SeeFormat SP).

Comments:
Again, most useful to get the index but if you set it in !?MR0, you can adjust Elementals sensitivity, for example. The basic damage is calculated before the triggers work so changing the spell number does not change the damage.
