Useful programs

ERA package includes several useful programs that can be used to enhance gameplay experience of ease the life of mod makers. This page will give brief instructions how to use each program.
BinMagic
Designed to work with binary patches format that is compatible with all versions of ERA of utility and ExeBuilder.
Technical details of the program parameters are given in the file "binmagichelp.txt".
To simplify the work in the directory with the program are 3 bat-files: binary.bat, text.bat, diff.bat.
The first two accept as the one argument: a patch file, which will be optimized and transferred to, respectively, binary or text form. diff.bat is used to create patches by comparing the original and modified files. Takes three arguments:
- The original file;
- Changed file;
- A way to save patches.

To save as a binary patch should have the extension "*. bin", otherwise any other. All three bat-file creates a patch for the memory, which can be activated by placing the directory EraPlugins \ [BeforeWoG | AfterWoG].
The text format of the patch:
The document consists of a patch label with useful information (author, scope, description) addresses, which need to be written new data and the data in hexadecimal (0-9a-f).

Example:
; Author = Unknown
; Version = 1.0
; Not for commercial use
00887668 12345678
00401FC7 9090A090B2
 The addresses are separated by whitespace information. Each new address is placed in a separate line.
Dee
DEE stands for Dynamic Erm Editor
The program lets you open, edit and manage a large amount of scripts and upload them to the editor of the game's memory (00 .. 99), edit them in memory and even completely delete fscripts from there. Additional features include the ability to extract the script from the compressed form in one line.
DEFka
This program allows to replace defs of any standard creature to another (you can find many of them on the internet) and also allows to restore their original graphics at any time
EVME
This tool allows the user to change the values stored in global and local variables
ExeBuilder
This program takes all *.bin patches from it's working directory, applies them to the file In.exe and saves the modified file as Out.exe
RepMan
This tool replaces all entities of string1 to string do in input file.
Example:
repman Data\s\script00.erm v2391 v600
ResMan
Program for managing snd/vid/lod archives. Run without parameters for help
LodImport
imports specified file to LOD
Parameters: path to lod, path to file
LodExport
A tool to export files from lod archive
parameters: path to lod, filename in lod, path to extracted file
Objector
Allows to combine object (z[e]obj[c]ts.txt) with custom files properly.
Parameters: targer file, custom file [parameter - replace already existing files]
DefExplode
This tool breaks up def file to frames in bmp format
Parameters: path to def file, path to extract frames
Protip: By using LodImport, ResMan and your own *.bat scripts, you can avoid using h3wupd installer, which often becomes unstable and corrupts data files.
